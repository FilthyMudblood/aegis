Directory layout
================
This folder is the single source tree for the project (there is no separate public/ copy to maintain). Develop and run from this repository root.

Optional proprietary layer
==========================
To use production tuning locally, add these files under aegis_private/ (they are gitignored and will not be pushed):

  aegis_private/metabolism.py       — HypothalamusEngine (full formulas)
  aegis_private/sensory.py          — calculate_fact_error_x (optional; not wired into public ACC — S = b*y + t*TCI in aegis_acc.py)
  aegis_private/acc_logic.py        — ACC: b/t/k weights + calculate_compliance_deviation_y (y)
  aegis_private/amygdala_rules.py   — TCI / amygdala regex and keyword lists (module routing, hijack patterns, intent classifier)
  aegis_private/config_tuning.py    — HYPOTHALAMUS_PROFILES and INTERVENTION_THRESHOLDS

If any file is missing, imports fall back to the matching *_open.py module or aegis_config_open / aegis_amygdala_rules_open.

Never commit API keys: set OPENAI_API_KEY / OPENAI_API_BASE in the environment or via the Streamlit sidebar.

When publishing to GitHub: keep .gitignore at the repo root so proprietary files under aegis_private/ stay local-only; no second export folder is required.
